  • 1.Alert

    描述：提出框，提供一个msg参数用于显示需要提示的信息。
    参数：msg:"" 需要显示的信息
    用法：layer.alert("提示消息");

  • 2.Confirm

    描述：确认提示框，参数详见用法。
    参数：{
    shade:[0.3,"#000"] 遮罩层样式设置,分别为opacity:透明度,background-color:背景颜色
    mask:true 是否需要遮罩,默认为true需要遮罩
    area:["310px","auto"] 提示框的大小，auto为根据内容自适应
    btns:1 配置显示按钮的数量,0:不显示按钮 1:显示一个确定按钮 2:显示一个确定按钮和取消按钮
    msg:"" 提示框需要显示的信息
    yes:function(){} 点击确定后的回调
    no:function(){} 点击取消后的回调
    }
    用法：layer.confirm({
    btns: 1,
    msg: "这个是一个确认提示框。当然我可以显示的信息是很多的，也可以设置我的按钮个数等等。详细参考参数说明。",
    yes: function () {
    alert("你点击了确定。");
    }
    });
    点击查看效果：

  • 3.Tips

    描述：信息提示框
    参数：{
    msg:"" 需要提示的信息
    time:3 自动延时关闭,单位为秒,0为不自动关闭
    position:{left:0,top:0,align:"center"} 消息提示框的位置,若设置了left或top则根据设置的值进行展示，否则center:代表居中 rightbottom:右下角
    }
    用法：layer.tips({
    msg: "我是一个消息提示框，一般用在错误或者其他的一些提示信息什么.",
    position: {align:"center" },
    time:3
    });
    点击查看效果：

  • 4.Iframe

    描述：iframe弹出窗
    参数：{
    mask: true, 是否需要遮罩,默认为true需要遮罩
    src: "http://www.baidu.com", 需要加载的URI
    area: ["910", "600"] 弹出窗的大小，分别为width和height
    }
    用法：layer.iframe({
    mask: true,
    src: "http://www.baidu.com",
    area: ["910", "600"]
    });
    点击查看效果：

  • 5.Html

    描述：html弹出窗
    参数：{
    html:"" 需要展示的html信息或dom
    }
    用法：layer.html({
    html:$("#hidden_info").show()
    });
